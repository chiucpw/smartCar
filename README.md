# 智慧汽車
智能汽車是一種高度程式化的汽車，具有普及計算、直觀的人機交互和開放的應用平台。 在這裡，我們提出了一個先進的智慧汽車演示平台，該平台具有透明擋風玻璃顯示螢幕和各種運動傳感器，司機可以在擴增實境中操控各種適合汽車的應用程式。 與智慧型手機類似，司機可以根據自己的需求免費下載適合汽車的應用程序來定制他們的智慧汽車。
